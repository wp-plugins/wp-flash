Allows to easily insert Adobe Flash animation into WordPress posts.

== Description ==

The WP-Flash plugin allows one to easily insert flash animation into WordPress blogs, using the following pseudo-code:

[swf:url width height]

where:
<ul>
    <li>url - URL of the flash object (SWF file) which you want to embed;</li>
    <li>width - width of the flash object;</li>
    <li>height - height of the flash object.</li>
</ul>

== Installation ==

1. Upload 'wp-flash.php' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
